# 基于机器学习的分布式系统的故障诊断网站项目
## 准备工作
```
1. 安装node.js环境
2. 安装依赖：npm install
```
## 启动应用
```
npm run dev
```
## 前端项目主要结构大纲
```
├── build ----------------------------------- 项目构建(webpack)相关配置文件，用于配置参数
├── mock ------------------------------------ 模拟接口，跟后端同步开发，加快开发进度
├── node_modules ---------------------------- npm 加载的项目依赖模块
│   ├── 略...
├── public----------------------------------- 公共文件
├── src ------------------------------------- 源代码
│   ├── api --------------------------------- 各种api组件
│   ├── assets ------------------------------ 静态文件，放置一些图片，如logo等
│   ├── components -------------------------- 组件目录，存放组件文件，未用
│   ├── icons ------------------------------- 一些矢量图文件
│   ├── layout ------------------------------ 布局文件
│   ├── router ------------------------------ 路由
│   ├── router ------------------------------ 层级路由文件
│   ├── store ------------------------------- 与存储相关文件
│   ├── style ------------------------------- 样式，主要是css代码
│   ├── utils ------------------------------- 工具文件
│   ├── view -------------------------------- 界面文件（核心代码）
│   ├── main.js ----------------------------- 主js
│   ├── App.vue ----------------------------- 项目入口组件
│   ├── permission.js ----------------------- 登录许可文件
│   ├── settings.js ------------------------- 设置文件
├── test ------------------------------------ 一些测试代码
├── package.json ---------------------------- node配置文件，记载着一些命令和依赖还有简要的项目描述信息 
├── babel.config.js ------------------------- babel的配置文件
├── package-lock.json ----------------------- 包版本控制文件
├── .README.md------------------------------- 项目的说明文档
├── .xxxx文件 -------------------------------- 一些配置文件，包括语法配置，git配置等
```

## 后端项目主要结构大纲
```
├── core ----------------------------------- 核心函数
├── Data ----------------------------------- 一些数据预处理代码
├── model ---------------------------------- 模型存储文件
├── pred------------------------------------ 一些预测测试代码
├── static --------------------------------- 静态文件
├── train ---------------------------------- 一些训练测试代码
├── uploads -------------------------------- 已上传的文件数据
├── app.py --------------------------------- 主要后端代码
├── main.py -------------------------------- main函数，未使用
```
